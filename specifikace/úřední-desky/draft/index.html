Reprezentace v otevřených formátech

Soubory s datovými schématy si můžete stáhnout, zahrnout do svých publikačních plánů a upravit dle vlastních potřeb.

JSON

Schéma pro formát JSON (JSON Schema)

XML

Schéma pro formát XML (XSD, XML Schema)

CSV

Je také možné použít zjednodušený zápis v CSV. Ten má ale jistá omezení. Nelze jej použít na prvky, které jsou poli. Např. pomocí formátu CSV nelze zapsat, že oznámení spadá pod několik agend, má více dokumentů (příloh) nebo se týká více subjektů.

Soubor v CSV je vhodné doplnit metadaty dle specifikace CSV on the Web.

Jména sloupců se tvoří z atributů, zanořování se řeší spojením názvů pomocí "_". Např. organizace_ico.

Příklady

Jednoduché oznámení

Všechny atributy schématu jsou nepovinné. Nicméně bez některých základních atributů publikace prakticky ztrácí smysl.

Příklad minimalistického oznámení zveřejněného na úřední desce: 

[
    {
        "id": 1024,
        "identifikator": "2018-13",
        "organizace": {
            "ico": "00258245"
        },
        "url": "https://example.com/uredni_deska/2018-2",
        "nazev": "Podpora spolkového života ve městě",
        "zverejneno": "2018-01-20",
        "sejmuto": "2018-02-20",
    }
]

Jednoduché oznámení s vlastními atributy

Do schématu si lze přidávat libovolně další vlastní atributy.

Příklad přidání vlastních atributů nad rámec standardu: 

[
    {
        "id": 1025,
        "identifikator": "2018-14",
        "organizace": {
            "ico": "00258245"
        },
        "url": "https://example.com/uredni_deska/2018-3",
        "nazev": "Podpora spolkového života ve městě II",
        "zverejneno": "2018-01-20",
        "sejmuto": "2018-02-20",
        "zverejnil": "Mirek Dušín",
        "odbor": "VD"
    }
]

Zápis v XML

Obdobně lze oznámení zaznamenat také v XML. Opět je zde možné, jako ukázané výše u JSON, si vybrat jen některé atributy nebo naopak další vlastní přidat.

Příklad zveřejnění dvou oznámení - jednoduchého a komplexního - v XML: 

<?xml version="1.0" encoding="UTF-8" ?>
<uredni_deska>
    <oznameni>
        <id>1024</id>
        <identifikator>P2018-12</identifikator>
        <organizace>
            <ico>00258245</ico>
        </organizace>
        <url>https://example.com/uredni_deska/2018-2</url>
        <nazev>Podpora spolkového života ve městě</nazev>
        <zverejneno>2018-01-20</zverejneno>
        <sejmuto>2018-02-20</sejmuto>
    </oznameni>
    <oznameni>
    	<id>1025</id>
    	<identifikator>P2018-13</identifikator>
    	<organizace>
    		<nazev>Město Horní Datová</nazev>
    		<ico>00258245</ico>
    		<adresa>
    			<kod_adm>696285</kod_adm>
    			<kod_obce>CZ0325 559351</kod_obce>
    			<nazev_obce>Horní Datová</nazev_obce>
